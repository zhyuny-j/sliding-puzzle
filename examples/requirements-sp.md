사진을 업로드하면 주어진 사진을 이용해 Sliding puzzle 을 만드는 서비스

## 화면 구성
- Title : 내가 원하는 사진으로 퍼즐 만들기
- 왼쪽 : 기본 퍼즐 및 URL 을 통해 사진 추가 버튼
- 가운데 : 주어진 사진을 통해 4X4 슬라이딩 퍼즐 제공
- 오른쪽 : 성공 기록(Ranking)
- 버튼 : 슬라이딩 퍼즐 하단에 Reset 아이콘 제공
- 테마 : 레트로 게임 스타일로 화면을 구성

## 사용자 흐름
1. 이미지 선택
  - 왼쪽에서 사진 선택 또는 URL 을 추가하여 이미지 추가
  - 이미지 추가시 이미지 이름 설정 가능
  - 이미지 선택 시 슬라이딩 퍼즐 4X4 가 나타남
2. 슬라이딩 퍼즐
  - 선택된 슬라이딩 퍼즐이 4X4 형태로 나타남
  - 마지막 16번째 이미지는 공백으로 설정
  - 퍼즐 조각을 랜덤으로 섞음
  - 사용자가 퍼즐을 클릭하면 공백으로 이동. 주위에 공백이 없는 경우 이동 불가
  - 15개의 조각이 모두 제 위치에 있으면 Success 화면을 출력하고 아이디 입력 가능
  - 아이디가 입력되면 오른쪽 Ranking 화면에 빨리 문제를 해결한 순서로 아이디와 소요 시간을 표시
3. 리셋버튼
  - 리셋버튼을 클릭하면 슬라이딩 퍼즐을 다시 섞음
  - 소요시간은 초기화됨
