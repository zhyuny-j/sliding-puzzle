## 개요

사용자가 기본 제공 이미지 중 하나를 고르거나 URL로 이미지를 추가해서, 그 이미지로 4X4 슬라이딩 퍼즐(15-puzzle)을 즉시 플레이할 수 있는 서비스. 퍼즐을 완성하면 아이디와 소요 시간이 해당 이미지의 랭킹에 기록된다.

## 범위

### 포함
- 화면 타이틀 "내가 원하는 사진으로 퍼즐 만들기" 표시
- 왼쪽 패널: 기본 프리셋 이미지 목록 + URL로 이미지 추가(이름 지정 가능)
- 가운데 패널: 선택된 이미지의 4X4 슬라이딩 퍼즐 플레이 (빈칸, 셔플, 이동, 완성 판정, 소요 시간 표시)
- 오른쪽 패널: 선택된 이미지에 대한 랭킹(아이디 + 소요 시간)
- 퍼즐 하단 Reset 버튼
- 레트로 게임 스타일 테마
- 추가한 이미지 목록과 랭킹 기록은 같은 브라우저로 재방문해도 유지됨

### 제외
- 로그인/계정 시스템 — 랭킹의 "아이디"는 인증 없는 자유 입력 닉네임
- 기기 로컬 파일 업로드 — 이미지 추가는 URL 입력과 기본 프리셋 선택만 지원 (사용자 확인 완료)
- 4X4 이외의 퍼즐 크기 (3X3, 5X5 등)
- 여러 사용자가 같은 퍼즐을 실시간으로 동시에 푸는 멀티플레이 기능
- 기기·브라우저 간 랭킹 공유 — 랭킹과 이미지 목록은 이 기기의 이 브라우저 안에서만 유지되고 다른 기기·브라우저로는 넘어가지 않음

## 시나리오

### S1. 첫 진입 화면
- **Given**: 사용자가 서비스에 처음 접속함 (아직 이미지를 선택하지 않음)
- **When**: 페이지가 로드됨
- **Then**
  - [x] S1-1 화면 상단에 타이틀 "내가 원하는 사진으로 퍼즐 만들기"가 표시된다
  - [x] S1-2 왼쪽 패널에 기본 프리셋 이미지 목록(4~6장, 제안 기본값)이 표시된다
  - [x] S1-3 가운데 패널에는 퍼즐 대신 이미지를 선택하라는 안내("이미지를 선택해주세요", 제안 기본값)가 표시된다
  - [x] S1-4 오른쪽 랭킹 패널에는 기록이 없다는 안내("아직 기록이 없습니다", 제안 기본값)가 표시된다

### S2. 기본 프리셋 이미지로 퍼즐 시작
- **Given**: 왼쪽 패널에 기본 프리셋 이미지 목록이 표시되어 있음
- **When**: 사용자가 프리셋 이미지 하나를 클릭
- **Then**
  - [x] S2-1 가운데 패널에 해당 이미지로 만들어진 4X4 퍼즐이 나타난다
  - [x] S2-2 16번째(마지막) 조각 자리는 빈칸으로 표시된다
  - [x] S2-3 조각 배치는 정해진 정답 배치와 다르게 섞인 상태로 나타난다 (완성 상태로 시작하지 않음)
  - [x] S2-4 퍼즐이 화면에 나타나는 즉시 소요 시간 표시가 00:00(제안 기본값 형식, INV-4)부터 흐르기 시작한다 (사용자의 첫 조작을 기다리지 않음, 제안 기본값)

### S3. URL로 이미지 추가
- **Given**: 왼쪽 패널에 이미지 URL과 이름을 입력하는 추가 폼이 보임
- **When**: 사용자가 유효한 이미지 URL과 이름("바다 풍경")을 입력하고 추가
- **Then**
  - [x] S3-1 왼쪽 이미지 목록에 이름 "바다 풍경"으로 새 항목이 추가된다
  - [x] S3-2 추가된 이미지가 자동으로 선택되어 가운데 패널에 해당 이미지의 4X4 퍼즐이 나타나고 소요 시간이 00:00부터 흐르기 시작한다 (S2-4와 동일)

### S4. 잘못된 URL로 이미지 추가 시도
- **Given**: 이미지 추가 폼이 보임
- **When**: 사용자가 이미지로 로드되지 않는 URL(깨진 링크 등)을 입력하고 추가를 시도
- **Then**
  - [x] S4 "이미지를 불러올 수 없습니다"(제안 기본값) 메시지가 나타나고, 목록에 새 항목이 추가되지 않는다

### S5. 셔플은 항상 풀 수 있는 상태로만 나온다
- **Given**: 임의의 이미지가 선택되어 퍼즐이 새로 섞인 직후
- **When**: 사용자가 규칙대로(빈칸에 인접한 조각만 이동) 계속 조작함
- **Then**
  - [ ] INV-1 조작을 반복하면 15개 조각이 모두 정답 위치에 오는 배치에 반드시 도달할 수 있다 (풀 수 없는 셔플 배치는 나오지 않는다)

### S6. 빈칸에 인접한 조각을 클릭하면 이동한다
- **Given**: 퍼즐이 섞인 상태로 진행 중
- **When**: 사용자가 빈칸과 상하좌우로 인접한 조각을 클릭
- **Then**
  - [x] S6-1 클릭한 조각이 빈칸 위치로 이동하고, 그 조각이 있던 자리가 새 빈칸이 된다

### S7. 빈칸에 인접하지 않은 조각을 클릭하면 이동하지 않는다
- **Given**: 퍼즐이 섞인 상태로 진행 중
- **When**: 사용자가 빈칸과 인접하지 않은 조각을 클릭
- **Then**
  - [x] S7 클릭한 조각은 움직이지 않고 배치가 그대로 유지된다

### S8. 퍼즐 완성 시 Success 화면과 아이디 입력
- **Given**: 사용자의 조작으로 15개 조각이 모두 정답 위치에 놓임 (빈칸도 정위치)
- **When**: 마지막 조각이 정위치로 이동하는 순간
- **Then**
  - [x] S8-1 완성된 원본 이미지와 함께 Success 화면이 나타난다
  - [x] S8-2 Success 화면에 최종 소요 시간이 표시되고, 소요 시간 표시가 그 순간에 멈춘다
  - [x] S8-3 아이디를 입력할 수 있는 입력창이 나타난다
  - [x] S8-4 완성 후에는 퍼즐 조각을 클릭해도 더 이상 움직이지 않는다

### S9. 아이디 제출 시 랭킹 반영
- **Given**: S8 완성 직후 Success 화면에서 아이디 입력창이 보임
- **When**: 사용자가 아이디("player1")를 입력하고 제출
- **Then**
  - [x] S9-1 오른쪽 랭킹 패널에 "player1"과 이번 소요 시간이 새 항목으로 나타난다
  - [x] S9-2 랭킹 항목은 같은 이미지에 대한 기존 기록들과 함께 소요 시간이 짧은 순서로 정렬되어 표시된다
  - [x] S9-3 랭킹은 현재 선택된 이미지에 대한 기록만 보여주고, 다른 이미지의 기록은 보이지 않는다 (이미지별 랭킹)

### S10. 빈 아이디로 제출 시도
- **Given**: S8 완성 직후 Success 화면에서 아이디 입력창이 비어있음
- **When**: 사용자가 아이디를 입력하지 않고 제출을 시도
- **Then**
  - [x] S10 제출이 되지 않고 "아이디를 입력해주세요"(제안 기본값) 안내가 나타난다 (아이디는 1~20자, 제안 기본값)

### S11. Reset 버튼으로 다시 섞기
- **Given**: 퍼즐이 진행 중이거나 완성된 상태
- **When**: 사용자가 Reset 버튼을 클릭
- **Then**
  - [x] S11-1 같은 이미지로 퍼즐이 새로 섞인다 (풀 수 있는 배치로, INV-1 유지)
  - [x] S11-2 소요 시간 표시가 00:00으로 초기화되고 즉시 다시 흐르기 시작한다
  - [x] S11-3 Success 화면이 표시되어 있었다면 사라지고 다시 조작 가능한 퍼즐 화면으로 돌아온다

### S12. 진행 중 다른 이미지로 전환 시 확인
- **Given**: 퍼즐을 완성하지 못한 채 진행 중
- **When**: 사용자가 왼쪽 목록에서 다른 이미지를 클릭
- **Then**
  - [x] S12-1 "진행 중인 퍼즐을 포기하고 전환하시겠습니까?"(제안 기본값) 확인 다이얼로그가 나타난다
  - [x] S12-2 확인을 선택하면 선택한 새 이미지로 퍼즐이 새로 섞여 시작되고, 이전 진행 상황은 저장되지 않는다
  - [x] S12-3 취소를 선택하면 원래 진행 중이던 퍼즐이 그대로 유지된다

### S13. 재방문 시 데이터 유지
- **Given**: 사용자가 URL로 이미지를 추가했고, 최소 1개 이미지에 랭킹 기록이 있는 상태에서 브라우저를 닫음
- **When**: 같은 브라우저로 서비스에 다시 접속
- **Then**
  - [ ] S13-1 왼쪽 목록에 이전에 추가한 이미지가 그대로 남아있다
  - [ ] S13-2 해당 이미지를 선택하면 오른쪽 랭킹 패널에 이전 기록이 그대로 표시된다

## 불변 규칙

- [x] INV-1 (S5에서 정의) 모든 셔플 결과는 항상 풀 수 있는 배치여야 한다
- [x] INV-2 세로/가로 비율이 정사각형이 아닌 이미지는 중앙을 기준으로 정사각형 크롭한 뒤 4X4로 분할한다
- [ ] INV-3 랭킹 기록과 추가된 이미지 목록은 새로고침·재방문 후에도 같은 브라우저에서 유지된다 (S13)
- [x] INV-4 화면에 표시되는 소요 시간은 항상 mm:ss 형식이다 (제안 기본값, 예: "01:23")
- [x] INV-5 랭킹 패널은 이미지별로 상위 10개(제안 기본값) 기록까지만 표시한다

## End-to-end 검증

1. 서비스 첫 진입 → 타이틀과 왼쪽 기본 프리셋 목록, 빈 랭킹 안내가 보인다 (S1)
2. 프리셋 이미지 하나를 클릭 → 가운데에 섞인 4X4 퍼즐이 나타나고 소요 시간이 00:00부터 흐른다 (S2)
3. 빈칸에 인접한 조각을 순서대로 클릭해 퍼즐을 완성 → Success 화면과 최종 소요 시간이 나타난다 (S6, S8)
4. 아이디를 입력하고 제출 → 오른쪽 랭킹에 아이디와 시간이 추가된다 (S9)
5. Reset 버튼 클릭 → 퍼즐이 다시 섞이고 시간이 00:00부터 다시 시작한다 (S11)
6. URL로 새 이미지를 이름과 함께 추가 → 목록에 추가되고 그 이미지로 퍼즐이 시작된다 (S3)
7. 페이지를 새로고침 → 추가한 이미지와 4단계에서 남긴 랭킹 기록이 그대로 남아있다 (S13)
